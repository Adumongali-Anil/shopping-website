Return Process

1

Initiate Return

Log into your account and select the order containing the item(s) you wish to return.

2

Print Label

Print the prepaid return shipping label that we'll provide.

3

Package Items

Pack the items securely in their original packaging if possible.

4

Ship Return

Drop off the package at any authorized shipping location.

Return Policy

Items must be returned within 30 days of delivery

Items must be unused and in original packaging with tags attached

Refunds will be processed within 5-7 business days after receiving the return

Original shipping charges are non-refundable

Sale items are final sale and cannot be returned
